1. 메세지 규약: Part Number / Question Number / Question Name

    - ex) part3_#5_bowlingball.py

2. 파일 형식: .py(문제 당 1개)

3. 주석: 문제 링크, 사용한 메모리, 걸린 시간 등

    - ex) #link: https://www.acmicpc.net/problem/1439
    - ex) #memory: 30840KB
    - ex) #time: 76ms
